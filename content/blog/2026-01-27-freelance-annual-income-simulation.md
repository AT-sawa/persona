「フリーコンサルになったら年収はどうなるのか」——独立を検討する際、最も気になるのがこの問いです。

結論から言えば、ファーム時代のマネージャー以上であれば、年収が上がるケースが多いです。ただし、単価だけで年収は決まりません。稼働率、空白期間、経費、税金——これらを含めたシミュレーションが必要です。

---

## 基本の計算式

フリーコンサルの年収は以下の式で算出されます。

**年収 = 月額単価 × 稼働率 × 年間稼働月数 − 経費 − 税金・社会保険料**

PERSONAの案件単価は100万〜250万円/月です。この単価帯をベースに、具体的なシミュレーションを行います。

## シミュレーション：4つのケース

### ケース1：IT PMO、稼働率100%

| 項目 | 金額 |
|---|---|
| 月額単価 | 130万円 |
| 稼働率 | 100% |
| 年間稼働月数 | 11ヶ月（1ヶ月の空白を想定） |
| 年間売上 | 1,430万円 |
| 経費（交通費、通信費、税理士等） | ▲100万円 |
| 所得 | 1,330万円 |

ファーム時代のマネージャー年収が800〜1,200万円程度だとすれば、税引前ベースで上回る可能性が高い。

### ケース2：業務改革、稼働率80%

| 項目 | 金額 |
|---|---|
| 月額単価 | 160万円 |
| 稼働率 | 80% |
| 年間稼働月数 | 12ヶ月（空白期間なし） |
| 年間売上 | 1,536万円 |
| 経費 | ▲120万円 |
| 所得 | 1,416万円 |

稼働率80%にすることで空白期間のリスクを減らし、年間通して安定した収入を得るパターン。残りの20%で次の案件探しや自己研鑽に充てられます。

### ケース3：メイン60%＋サブ20%の複数案件

| 項目 | 金額 |
|---|---|
| メイン案件 月額単価 | 150万円（稼働率60%＝90万円/月） |
| サブ案件 月額単価 | 100万円（稼働率20%＝20万円/月） |
| 合計月額 | 110万円 |
| 年間稼働月数 | 12ヶ月 |
| 年間売上 | 1,320万円 |
| 経費 | ▲130万円 |
| 所得 | 1,190万円 |

単価は下がるが、2つの案件を持つことで収入源を分散し安定性を確保するパターン。

### ケース4：戦略アドバイザリー、稼働率40%

| 項目 | 金額 |
|---|---|
| 月額単価 | 200万円（稼働率40%＝80万円/月） |
| 年間稼働月数 | 12ヶ月 |
| 年間売上 | 960万円 |
| 経費 | ▲80万円 |
| 所得 | 880万円 |

稼働率を抑えて時間の自由度を最大化するパターン。残りの60%で別の案件を追加すれば、年収はさらに上がります。

## ファーム時代との比較

| 項目 | ファーム（マネージャー） | フリーコンサル |
|---|---|---|
| 年収レンジ | 1,000〜1,500万円 | 1,200〜2,500万円（稼働率と単価による） |
| ボーナス | あり（業績連動） | なし（毎月が固定収入） |
| 福利厚生 | 充実 | 自費 |
| 退職金 | あり | なし（法人化すれば積立可） |
| 社会保険 | 会社が半分負担 | 全額自己負担 |
| 経費 | 会社負担 | 自己負担（ただし経費計上可） |

単純な額面比較ではフリーコンサルが有利に見えますが、福利厚生、退職金、社会保険の会社負担分を含めると、ファームの実質報酬はもう少し高くなります。これらを考慮しても、フリーコンサルの年収がファーム時代を上回るケースは多いですが、正確な比較のためには税理士への相談を推奨します。

## 法人化による手取り最適化

フリーコンサルとして一定の売上規模に達したら、法人化（マイクロ法人の設立）を検討する価値があります。法人化には初期費用や手間がかかりますが、手取りを最適化する上で有効な手段です。

### 法人化を検討すべきタイミング

年間売上が1,000万円を超えるタイミングが一つの目安です。消費税の課税事業者になる基準額であり、個人事業主のままでは税負担が急増するポイントでもあります。フリーコンサルの場合、月額単価が100万円以上であれば、多くのケースで初年度から1,000万円を超えます。

### 法人化のメリット

**役員報酬による所得分散。** 法人から自分に支払う役員報酬を調整することで、個人の所得税率を抑えられます。年間売上1,500万円の場合、個人事業主のままでは所得税率33%の区分に入る可能性がありますが、法人化して役員報酬を適切に設定すれば、法人税と個人の所得税を合わせた実効税率を下げることが可能です。

**社会保険料の最適化。** 役員報酬の水準に応じて社会保険料が決まるため、報酬額の設計によって社会保険料の負担を調整できます。

**経費の幅が広がる。** 法人の場合、個人事業主では認められにくい支出（出張手当、社宅費用、生命保険の一部など）を経費として計上できる場合があります。

**退職金の積立。** 法人であれば小規模企業共済や中小企業退職金共済に加え、法人としての退職金制度を設計できます。ファーム時代にあった退職金に相当する仕組みを自分で構築できるのです。

### 法人化の注意点

法人の設立費用（登録免許税、定款認証費用など）で20〜30万円程度が必要です。また、決算・税務申告が複雑になるため、税理士との顧問契約が実質的に必須となります。年間で30〜50万円の税理士費用を見込んでおく必要があります。法人化の判断は個別の状況によって大きく異なるため、必ず税理士に相談してください。

## 経費と税金の現実的な見積もり

シミュレーションの精度を上げるためには、経費と税金の内訳を理解しておく必要があります。

### フリーコンサルの主な経費項目

| 経費項目 | 年間目安 | 備考 |
|---|---|---|
| 交通費 | 20〜40万円 | 常駐案件か、リモート案件かで大きく変動 |
| 通信費・IT環境 | 10〜20万円 | PC、スマートフォン、インターネット回線、クラウドサービス |
| 税理士・会計費用 | 30〜50万円 | 法人の場合は必須、個人でも確定申告の外注を推奨 |
| 書籍・研修費 | 5〜15万円 | 専門性維持のための投資 |
| 交際費 | 10〜30万円 | ネットワーキング、情報交換のための会食 |
| オフィス関連 | 0〜30万円 | コワーキングスペースの利用料など |
| 保険・共済 | 20〜40万円 | 所得補償保険、小規模企業共済など |

PERSONAの案件はリモートと常駐がほぼ半々の比率であるため、完全リモートであれば交通費を大幅に削減できます。一方、常駐案件が多い場合は交通費が膨らむため、案件選定の段階で考慮しておくべきです。

### 税金・社会保険料の概算

年間所得1,500万円の場合、所得税・住民税・事業税で約400〜500万円、国民健康保険・国民年金で約100〜120万円が目安です。法人化していれば社会保険料の設計により異なりますが、いずれにしても手取りは売上の50〜60%程度になることを前提としたシミュレーションが現実的です。

## 年収を最大化するための3つの鉄則

### 鉄則1：空白期間をゼロにする

月額150万円の案件で年間2ヶ月の空白期間があると、300万円の機会損失です。PERSONAの案件平均期間は約2年と長期ですが、案件終了前に次の案件を確保する運用が重要です。PERSONAでは参画中の案件終了を見据えて、事前に次の案件を提案する体制を取っています。

### 鉄則2：適正単価を維持する

不安から単価を下げると、それが「あなたの相場」として定着します。PERSONAのエージェントはファーム出身であり、独自マッチングアルゴリズムと合わせて、あなたの経験に対する市場の適正評価を具体的な数字でお伝えできます。案件延長率約9割、再依頼率ほぼ100%のため、安定した年収計画が立てやすい環境です。

### 鉄則3：稼働率を戦略的に設計する

100%稼働が必ずしも年収最大化にはつながりません。80%稼働で空白期間ゼロのほうが、100%稼働で年2ヶ月空白よりも年収が高くなることもあります。

PERSONAでは登録者1,200人以上のMBB・Big4・アクセンチュア出身者に対して、30社以上の提携エージェントの案件を含めて常時100件以上の中から最適な稼働率設計を提案しています。

## 独立初年度の資金計画

年収シミュレーションと並んで重要なのが、独立初年度の資金計画です。初年度は想定外の出費が発生しやすく、資金ショートのリスクを正確に把握しておく必要があります。

### 独立前に確保すべき生活費

最低6ヶ月分の生活費を手元に確保してから独立することを推奨します。案件が決まるまでの期間、案件開始後の初回入金までのタイムラグ（通常1〜2ヶ月）、そして万が一案件が早期終了した場合のバッファーを考慮すると、6ヶ月分は最低ラインです。

### 初年度特有のコスト

独立初年度には、法人設立費用（法人化する場合）、業務環境の整備（PC、ソフトウェア、名刺等）、各種保険の加入手続きなど、ファーム時代には発生しなかったコストがかかります。これらを合計すると、50〜100万円程度の初期投資を見込んでおくべきです。

### 初回入金までのタイムラグ

見落としがちなのが、案件参画から初回の報酬を受け取るまでのタイムラグです。多くの場合、月末締め翌月末払いの契約となるため、案件参画の翌月末まで報酬が入りません。4月に独立して即座に案件に参画したとしても、最初の入金は5月末になるケースが一般的です。この1〜2ヶ月間を乗り切る資金は、独立前に確保しておく必要があります。

PERSONAでは登録から参画まで最短即日対応を行っているため、タイムラグを最小限に抑えることが可能です。ファーム出身のエージェントが、資金計画も含めた独立準備の相談に対応しています。
